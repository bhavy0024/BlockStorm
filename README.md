# 🎮 BlockStorm

**BlockStorm** is a **Tetris-inspired console game** developed in **C++** using **Object-Oriented Programming (OOP)**. It combines classic Tetris gameplay with modern features like **Ghost Piece, Hold Piece, Next Piece Preview, Hard Drop, Wall Kick Rotation, and High Score Saving**. This project highlights modular game development, collision detection, console rendering, and file handling in C++.

---

## ✨ Features

- 🎲 7 Tetrominoes
- 🎨 Colored Tetrominoes
- ⬅️➡️ Smooth Left/Right Movement
- 🔄 Clockwise & Counter-Clockwise Rotation
- 🧱 Wall Kick Rotation
- 👻 Ghost Piece
- 📦 Hold Piece
- 🔮 Next Piece Preview
- ⚡ Hard Drop
- 🗑️ Line Clearing
- 📈 Level & Dynamic Speed Increase
- 🏆 Score & High Score Saving
- ⏸️ Pause / Resume
- 🔁 Restart Game

# 🚀 Installation Guide

Follow these steps to install and run **BlockStorm** on your machine.

## 1️⃣ Clone the Repository 🖥️

```sh
git clone https://github.com/bhavy0024/BlockStorm.git
```

## 2️⃣ Navigate to the Project Directory 📂

```sh
cd BlockStorm
```

## 3️⃣ Compile the Source Code ⚙️

Using **g++ (MinGW)**:

```sh
g++ main.cpp -o BlockStorm
```

## 4️⃣ Run the Game 🎮

### Windows 🪟

```sh
BlockStorm.exe
```

or

```sh
.\BlockStorm.exe
```

> **Note:** This project currently uses `windows.h` and `conio.h`, so it runs only on Windows.

## 5️⃣ Update to the Latest Version 🔄

```sh
git pull origin main
```
